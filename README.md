# continuous-alarm

## What is it?

株式会社プログメイト様のインターンに参加させていただき、2日間で作成したアラームアプリです。

React Native, Expoを使用しています。

デザインはReact Native Elementsを使用しています。

時間選択機能と、曜日選択機能があります。
